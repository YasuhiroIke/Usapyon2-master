# Usapyon2-master

「うさぴょん２」ではクラスタ並列化構成を予定しているのですが、その内のマスター側部分になります。

元にしているソフトは基本的にありません。

「なのはmini」の作者、川端一之様、「Apery」の作者、平岡拓也様、 「StockFish」の作者、Tord Romstad様、Marco Costalba様, Joona Kiiski様,Gary Linscott様に深く感謝をしております。
